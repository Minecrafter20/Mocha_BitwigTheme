# Mocha

# Instalacion 
- Follow these steps: [Berikai](https://github.com/Berikai/bitwig-theme-editor.git)
- Download this archive: [`mocha.json`](mocha.json)

# Capturas
Configuration of captures in Bitwig
- Half tones: 0%
- Black level: 85%
- Grid lines: 50%
